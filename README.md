# CLI_Calculator
A command line interface calculator built in C to test to learn arrays functions and such.


How To Compile:
gcc main.c moha_math.c moha_utils.c -o name_prog

How To Run:

Powershell = name_prog


Bash = ./name_prog


Features:

1.Simple arithmic operations (Addition/Subtraction/Multiplication/Division)

2.Factorial and power of a number.

3.Even, odd, and prime checker.
